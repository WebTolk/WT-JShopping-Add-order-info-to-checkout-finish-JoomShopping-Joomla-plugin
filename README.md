# WT JShopping Add order info to checkout finish Joomla plugin
## RU
Плагин отображения информации о заказе на странице "Спасибо за заказ" интернет-магазина JoomShopping.
Вы можете создать свой собственный макет вывода информации о заказе в папке `plugins/jshoppingorder/wtjshoppingaddorderinfotocheckoutfinish/tmpl`.
## EN
Plugin for displaying order information on the "Thank you for your order" page of JoomShopping online store.
You can create your own layout in  `plugins/jshoppingorder/wtjshoppingaddorderinfotocheckoutfinish/tmpl` folder.
